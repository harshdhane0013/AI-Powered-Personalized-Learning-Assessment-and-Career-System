# AI-Powered-Personalized-Learning-Assessment-and-Career-System
AI-powered web application that generates personalized learning paths, quizzes, and performance analytics using Flask and AI APIs.

# AI-Powered Personalized Learning System

## Features
- Personalized Learning Path Generator
- AI-Based Study Notes
- Automatic Quiz Generator
- Performance Tracking
- Parent Portal

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite Database
- OpenRouter API
- YouTube API

## How to Run
1. Install dependencies
2. Run server.py
3. Open browser at http://localhost:5000
